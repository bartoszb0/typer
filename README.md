# Typer

**Typer** is a lightweight web-based utility for testing and improving your typing speed and accuracy.  
It provides feedback on your words-per-minute (WPM), accuracy, and typing performance — all in a clean, minimal interface.

> Try here
> https://typertyper.netlify.app/

## Overiew
- Start a test and type the given words as quickly and accurately as you can.
- Once the test ends, your results are displayed instantly — no downloads or accounts required.

## Technologies Used

- React
- JavaScript
- Vite
- Netlify

## Running Locally
```
git clone https://github.com/bartoszb0/typer
cd typer
npm install
```
```
npm run dev
```
